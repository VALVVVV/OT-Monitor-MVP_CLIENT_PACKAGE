КАК ПОДГОТОВИТЬ КОМПЬЮТЕР ДЛЯ ОТ-МОНИТОР

1. Что нужно для работы ОТ-Монитор

Для работы нужны:
- Windows-компьютер;
- Python 3 для Windows;
- библиотека requests;
- библиотека beautifulsoup4, в проверке она отображается как bs4;
- современный браузер Chrome или Edge;
- интернет для проверки внешних источников.

2. Как проверить готовность

1. Запустите CHECK_READY.bat.
2. Дождитесь окончания проверки.
3. После проверки появится файл CHECK_READY_RESULT.txt.
4. Этот файл можно отправить разработчику или техническому специалисту.

3. Если Python не найден

1. Скачайте Python 3 с официального сайта python.org.
2. Во время установки включите Add Python to PATH, если такая галочка есть.
3. После установки снова запустите CHECK_READY.bat.

4. Если requests или bs4 не найдены

1. Нажмите Win + R.
2. Введите cmd.
3. Нажмите Enter.
4. В открывшееся окно вставьте команду правой кнопкой мыши:

python -m pip install requests beautifulsoup4

5. Нажмите Enter.
6. Дождитесь окончания установки.
7. Снова запустите CHECK_READY.bat.

5. Если пользователь не хочет устанавливать сам

1. Отправьте техническому специалисту файл CHECK_READY_RESULT.txt.
2. Также отправьте ему этот файл КАК_ПОДГОТОВИТЬ_КОМПЬЮТЕР.txt.
