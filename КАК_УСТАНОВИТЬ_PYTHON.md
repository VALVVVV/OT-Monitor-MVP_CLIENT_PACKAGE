# Как установить Python для ОТ-Монитор

## 1. Когда нужна эта инструкция

Эта инструкция нужна, если файл `Проверить готовность.bat` показывает, что Python не найден на компьютере.

Для запуска ОТ-Монитор нужен Python 3 для Windows.

## 2. Где скачать Python

Скачивайте Python только с официального сайта:

```text
https://www.python.org/
```

Нужен Python 3 для Windows.

## 3. Что важно отметить при установке

Во время установки обратите внимание на галочку:

```text
Add Python to PATH
```

Если такая галочка есть, её нужно включить.

После этого завершите установку обычным способом.

## 4. Как проверить после установки

После установки снова откройте файл:

```text
Проверить готовность.bat
```

Если Python установлен правильно, проверка должна показать версию Python и перейти к следующему этапу проверки.

## 5. Что делать, если после установки Python всё равно не найден

Если Python уже установлен, но `Проверить готовность.bat` всё равно пишет, что Python не найден:
- закройте все старые окна командной строки;
- попробуйте открыть `Проверить готовность.bat` ещё раз;
- если проблема остаётся, лучше обратиться к техническому специалисту.

Обычно это означает, что Python установлен, но не был добавлен в PATH, или Windows ещё не обновила системный путь для текущего окна.
