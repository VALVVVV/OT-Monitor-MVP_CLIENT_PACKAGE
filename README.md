# OT-Monitor MVP

This repository contains the MVP version of OT-Monitor.

## Local launch

1. Run `CHECK_READY.bat`.
2. If the check is successful, run `START_OT_MONITOR.bat`.

## User instructions

The main user instructions are stored in:
- `НАЧНИТЕ_ОТСЮДА.txt`
- `КАК_ПОДГОТОВИТЬ_КОМПЬЮТЕР.txt`

## Review mode

The review mode saves the latest RTF report in:
- `CLIENT_OUTPUT/ZAMECHANIYA_DLYA_RAZRABOTCHIKA.rtf`

## Technical note

The technical Python files currently remain in the project root because they are used by the "Check new documents" workflow.
They should not be moved yet without updating the related paths.
